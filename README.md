### 项目运行

##### 温馨提示：由于该项目大量使用的ES6新特性，nodejs必须是较新的版本(6.0以上)，建议使用稳定版本

* git clone https://github.com/15826954460
* cd nodejs
* npm/cnpm install
* npm/cnpm run dev
